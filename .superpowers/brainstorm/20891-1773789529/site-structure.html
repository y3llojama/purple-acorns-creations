Site Structure

Two-zone architecture — public storefront + protected admin. Does this look right?

✦ Public Site
/ — Home
Hero with tagline · Featured pieces (3–4 products) · About teaser with link · Upcoming events strip · Instagram feed preview · CTA to shop
/shop — Shop
Embedded Square Online store iframe · Filter by category (rings, necklaces, earrings, bracelets, crochet) · Checkout handled by Square
/our-story — Our Story
Mother-daughter narrative · Rich text + photos · Brand values · How each piece is made
/events — Events
Upcoming arts & crafts fairs · Date, location, map link · Past events archive
/gallery — Gallery
Photo grid of pieces · Pulls from Instagram and/or uploaded images
/contact — Contact
Simple form · Instagram link · Custom order inquiry
⚙ Admin Panel (password protected)
/admin — Dashboard
Overview · Quick links to all sections · Live site preview
/admin/content — Content
Edit Our Story text + photos · Edit homepage hero text · Edit about blurb
/admin/events — Events
Add / edit / delete upcoming events · Date picker, location, notes
/admin/gallery — Gallery
Upload photos · Drag to reorder · Delete · Tag by category
/admin/branding — Branding
Switch theme: Warm Artisan ↔ Soft Botanical · Upload logo · Set accent color · Update social links
/admin/integrations — Integrations
Square Online store URL config · Instagram handle · Manage embed settings
Theme Switcher
Both Warm Artisan and Soft Botanical themes are built as CSS variable sets. Admin toggles from /admin/branding — change applies instantly site-wide with no rebuild needed.
